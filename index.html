About Me

I am Mayank R. Rathod, a final-year BSc Computer Science student and passionate Full Stack Developer with a strong foundation in both frontend and backend development. I specialize in creating clean, responsive, and user-friendly web applications, combining modern UI design with robust backend architecture. My technical expertise includes HTML, CSS, JavaScript (Vanilla) for frontend development and Node.js, Express.js, and Java Spring Boot for backend systems, with hands-on experience in MySQL for database management, queries, joins, and CRUD operations. I am skilled in implementing secure authentication using JWT and bcrypt, and have integrated AI-powered functionalities into projects using the OpenAI API. My portfolio includes projects such as a food delivery web application with admin and user management, a personal finance tracker in Python with data visualization, and a frontend AI chatbot for generating website code. Each project reflects my focus on clean code, performance optimization, and practical problem-solving. I thrive in environments that encourage learning, innovation, and collaboration. With a strong drive to keep up with emerging technologies like AI integration and advanced backend development, my goal is to build solutions that are functional, scalable, and impactful in real-world scenarios.
